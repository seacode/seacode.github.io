Code for (and by) fisheries stock assessment modelers

Welcome to Seacode

Seacode is a GitHub Organization for Fisheries Stock Assessment Modelers. The repository hosts a number of open-source projects currently under development. Each project can be accessed via its respective GitHub Project page listed below.

Current Projects

  • Gmacs: a generalized size-structured stock assessment modeling framework
  • Gmr: an R package in support of the Gmacs project
  • Cstar: a stock assessment function library for ADMB
  • Rstar: an R package providing complimentary code for the ADMB Cstar Library
